# UniCat
Eine Django-basierte Webplattform, auf der Künstler und Kunstbegeisterte einzigartige Kunstwerke versteigern und ersteigern können.

## Über das Projekt
Unicat ist eine Online-Auktionsplattform für einzigartige Kunstwerke. Egal ob Hobbykünstler, Profi oder einfach nur kunstbegeistert, bei Unicat bist du herzlich willkommen!

Künstler können ihre Werke zur Auktion einstellen
Käufer können einzigartige Kunstwerke durchstöbern und mitbieten
Reibungsloser Ablauf: Käufer teilen ihre Lieferadresse nach gewonnener Auktion
Bewertungssystem: Verkäufer können Käufer nach abgeschlossener Auktion bewerten

## Funktionen

Eigene Kunstwerke zur Auktion einstellen
Auf einzigartige Werke von Künstlern bieten
Adressaustausch zwischen Käufer und Verkäufer nach erfolgreicher Auktion
Bewertungssystem
PDF-Generierung für Auktionsdokumente
